# tamagotchi
Online tamagotchi pet

## About the project:
This is a cute, online tamagotchi pet that will die if you do not take proper care of it.
Like any regular pet, it has the following needs:
- hunger
- energy
- happiness
- hygiene

 There are meters that gradually decay over time, and if any of them fall to 0, your pet will die. Press the respective buttons labeled with the need to increase that specific meter. 
 If your pet dies, click the **reset** button

## Languages: 
Javascript, CSS, and html

Press a button and start playing

Enjoy!

(If you press the tamagotchi pet icon you'll get a surprise)
